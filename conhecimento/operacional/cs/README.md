## Princípio Fundamental

> **O CS não entrega resultado — o CS garante que o cliente execute para ter resultado.**

A plataforma entrega as ferramentas. O Athos adapta para cada clínica. O CS garante que o cliente use o que tem disponível e avance. Sem execução não há resultado. Sem resultado não há retenção.
